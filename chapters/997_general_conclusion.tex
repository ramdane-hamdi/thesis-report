\label{chap:general_conclusion}

\section*{Conclusion}
\begin{flushleft}
This work explored how intelligent methods can support polypectomy by combining
computer vision and deep reinforcement learning. The main takeaway is simple: it is not a
single-image classification problem, but a sequential decision problem where each
action changes what comes next.

The main contribution is a structured base for building adaptive systems that can assist
decision-making in visually complex settings. Future work could make the simulation more
realistic, use richer data, and connect the framework more closely to real endoscopic
workflows.
\end{flushleft}
